# Compose Material 3 for Wear OS

Source URL:
https://developer.android.com/develop/ui/compose/layouts/material-3

Scope
Developer documentation for Material 3 components in Wear OS Compose, including library coordinates and component list.

Key points
- Defines the Compose Material 3 library for Wear OS and the dependency coordinates.
- Lists core components such as buttons, cards, dialog, pickers, progress indicators, sliders, and toggle controls.
- Notes that Material 3 for Wear OS is in alpha, so APIs may evolve.

How to use
- Use when handing off expressive UI to engineers or validating component availability in Wear OS Compose.
